##  Stack Tecnológico (Enterprise AAA)
La UTAMV utiliza una arquitectura de vanguardia para garantizar latencia cero y soberanía total:

- **Frontend:** [Vite](https://vitejs.dev/) + [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
- **UI/UX:** [Tailwind CSS](https://tailwindcss.com/) + [shadcn/ui](https://ui.shadcn.com/) (Minimalismo Platino/Neon)
- **Backend:** [Supabase](https://supabase.com/) (PostgreSQL + RLS Zero-Trust)
- **IA Kernel:** Protocolo Isabella (Telemetría Cognitiva e Interfaz de Alianza)
- **Despliegue:** Vercel Edge Network

---

##  Estructura de las 7 Federaciones (MD-X4)
El código de este repositorio está segmentado según la gobernanza estratégica de TAMV Online:

1. **Federación Académica:** Lógica de educación basada en resultados (OBE NextGen 2026).
2. **Federación de Datos:** Telemetría en tiempo real a través de `tamvcrums_logs`.
3. **Federación de IA:** Orquestación del Kernel Isabella y diálogos de alianza.
4. **Federación de Seguridad:** Blindaje Zero-Trust y políticas RLS.
5. **Federación de Infraestructura:** Gestión de contenedores y despliegue en el Nodo RDM.
6. **Federación de Contenidos:** Activos visuales de alta gama para el aprendizaje inmersivo.
7. **Federación de Interconexión:** Sincronización con ORCID, Zenodo e ISNI para validación científica.

## Guía de Desarrollo para Aliados
Si eres un desarrollador humano o una IA autorizada, sigue estos pasos:

```sh
# 1. Clonar el repositorio de soberanía
git clone [https://github.com/OsoPanda1/utamv-academic-core.git](https://github.com/OsoPanda1/utamv-academic-core.git)

# 2. Navegar al núcleo
cd utamv-academic-core

# 3. Inyectar dependencias
npm install

# 4. Activar el entorno de desarrollo
npm run dev
````
